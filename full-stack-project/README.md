\# Full Stack Portfolio Website



\## Overview



This project is a full-stack portfolio website with an admin dashboard that allows dynamic content updates without modifying the source code.



It includes authentication, database integration, and a contact system.



\## Features



\- User-facing portfolio website

\- Admin dashboard with login system

\- Ability to update website content without editing code

\- Secure login with hashed passwords

\- Contact form that stores messages in SQL database



\## Tech Stack



\- Frontend: HTML, CSS, JavaScript

\- Backend: PHP

\- Database: SQL



\## Security



\- Passwords stored as encrypted hashes

\- Login authentication system implemented using PHP and SQL



\## Functionality



\### Admin Panel

\- Login securely

\- Modify website content dynamically

\- Manage stored data



\### Contact System

\- Users can send messages through the website

\- Messages are stored in the SQL database

\- Accessible through admin panel
