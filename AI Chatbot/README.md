\# AI Chatbot



\## Overview



AI Chatbot is a rule-based conversational program that interacts with users through text input and generates responses using predefined intents and patterns.



This project focuses on building a structured chatbot without machine learning, using pattern matching and organized intent data.



\## Features



\- Command-line chatbot interaction

\- Pattern-based intent matching (no ML)

\- Structured responses using JSON data

\- Easily extendable by adding new intents

\- Lightweight and fast execution



\## Tech Stack



\- Python

\- JSON



\## How It Works



1\. User enters input in the terminal

2\. Input is matched against predefined patterns

3\. Corresponding response is returned

4\. Loop continues until exit command
